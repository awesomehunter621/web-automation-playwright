# web-automation-playwright

Basic UI automation tests using Playwright (JavaScript).

## What it does
- Launches a browser
- Navigates to a website
- Verifies page elements and text

## Tech used
- Node.js
- Playwright

## How to run
1. Install Node.js (v18+)
2. Install dependencies:
   npm install
3. Install browsers:
   npx playwright install
4. Run tests:
   npx playwright test
